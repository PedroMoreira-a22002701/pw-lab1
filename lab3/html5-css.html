HTML5 e CSS


HTML5

  • HTML5 (Hypertext Markup Language) é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet.

Elementos Semânticos Usado em alguma página? Se sim, onde?
Header ✅ Todas, que indica o cabeçalho
Nav ✅ Todas, que indica o menu de navegação.
Main ✅ Todas, que indica o conteúdo principal
Article ❌ Nenhuma, indica a porção de um documento
Aside ❌ Nenhuma, indica anotações
Section ✅ Todas, especifica parte do conteúdo
Footer ✅ Todas, indica o rodapé
Details ❌ Nenhuma, define detalhes
Meter ❌ Nenhuma, define uma medida escalar estática
Progress ❌ Nenhuma, define uma barra de progresso de uma tarefa
Abbr ❌ Nenhuma, indica o significado de uma abreviatura.
Address ❌ Nenhuma, indica a informação sobre o contacto do autor

CSS

  • CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo, usada para descrever a apresentação de um documento escrito em HTML ou em XML.

Seletor CSS Usado em alguma página? Se sim, onde?
Seletor Universal(*) ❌ O seletor universal (*) aplica o estilo a todos os elementos da página.
Seletor Tipo ✅ Usado no info.html e html5-css.html.
O seletor de tipo usa o nome do marcador a estilizar.
Seletor de Classe ✅ Usado em info.html, html5-css.html e quizz.html.
O seletor de classe associa um nome a um estilo.
Seletor de id ❌ O atributo id permite identificar univocamente um elemento do HTML
Selector de Atributo ❌ Seletores que se aplicam a atributos de elementos.


Combinadores de seletores Foi usado? Se usado, onde?
div p ❌ Todos os p que estão “dentro” de um div.
div > p ❌ Todos os p que estão “logo dentro” de div.
div ~ p ❌ Todos os p que estão colocados depois de um div.
div + p ❌ Todos os p que estão colocados “imediatamente” depois de um div.
div,p{ color: red; } ❌ Os div e p ficacarão com fonte vermelha.
